
# Sunset Tracker CLI

A simple terminal tool to show sunset data for a chosen location.


---

## Installation

### Recommended: Using pipx (isolated CLI)

```bash
pipx install git+https://github.com/Gdev45/sunset-tracker.git

```
## Run using

```bash
sunset-tracker
```
